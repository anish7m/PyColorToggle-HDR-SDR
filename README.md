# PyColorToggle-HDR-SDR

PyColorToggle-HDR-SDR is a fork of the original PyAutoActions project by [7gxycn08](https://github.com/7gxycn08). It automates HDR management for games and applications by toggling HDR or SDR based on process status, and this fork adds the ability to apply/reload the current ICC color profile when HDR or SDR is toggled.

## Original Author Credit

This application is based on the original work by [7gxycn08](https://github.com/7gxycn08). Credit and thanks go to the original author for creating the base application and HDR automation workflow.

## Features

This fork adds support for applying/reloading the current ICC color profile when HDR or SDR is toggled. For a full overview of the base application features, please refer to the [original author's GitHub repository](https://github.com/7gxycn08/PyAutoActions).
